# **App Name**: Algorythm AI

## Core Features:

- AI-Powered Builder Selection: Landing page interface with a dropdown menu to select dApp, NFT, or Cryptocurrency builder, directing users to the appropriate AI-driven builder page.
- Authentication: User authentication system supporting Google, email, and GitHub login, secured with Firebase Authentication.
- Credit System and Subscription: Implements a credit-based system with initial free credits. Integrates Stripe and NOWPayments for purchasing credit packs and monthly subscriptions (40 credits/$25, 100 credits/$50, 250 credits/$100, unlimited access/$500).
- AI-Driven dApp Builder: Chat interface for dApp building using Gemini-2.5-Flash as a tool. Features an interactive live preview, code editor, Google Cloud/Firebase integration for subscribing users, blockchain deployment setup, and GitHub sync. Provides checklist steps.
- AI-Powered NFT Builder: NFT builder powered by "Nano Banana", enabling users to create single NFTs or entire collections with guided deployment and checklist steps.
- AI Cryptocurrency Builder: Cryptocurrency builder using Gemini-2.5-Flash as a tool with guided deployment system and checklist steps.
- Unified Navigation: Top-right hamburger menu on every page provides access to settings, contact, and other common web app functionalities.
- Legal and Social Footer: Production grade docs, privacy policy, and terms and conditions with buttons on the Footer along with Social buttons(Facebook, X, etc.

## Style Guidelines:

- Primary color: Electric blue
- Background color: Black
- Accent color: Neon green and yellow.
- Font pairing: 'Space Grotesk' for headlines (sans-serif) and 'Inter' for body text (sans-serif), ensuring readability while keeping a modern, tech-focused feel.
- Implement a clean and modern layout, using white space effectively to avoid clutter. Each builder page has a dedicated section for user input, AI responses, live preview, and code editing.
- Subtle animations and transitions when loading AI responses or toggling between different sections to provide a smooth user experience.
