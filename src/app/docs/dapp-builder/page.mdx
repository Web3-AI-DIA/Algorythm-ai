# dApp Builder

Our AI-driven dApp Builder helps you create decentralized applications with ease.

## Features

*   **AI-Powered Chat Interface**: The dApp Builder uses a chat interface powered by **Gemini-2.5-Flash** to guide you through the process of creating your dApp.
*   **Interactive Live Preview**: See a live preview of your dApp as you build it, allowing you to make changes and see the results in real time.
*   **Code Editor**: Access the underlying code of your dApp and make changes directly in the built-in code editor.
*   **Google Cloud/Firebase Integration**: Seamlessly integrate your dApp with Google Cloud and Firebase for subscribing users.
*   **Blockchain Deployment Setup**: The dApp Builder helps you set up the deployment of your dApp to the blockchain.
*   **GitHub Sync**: Sync your dApp's code with a GitHub repository to keep track of your changes and collaborate with others.
*   **Checklist Steps**: The dApp Builder provides a checklist of steps to guide you through the process of creating and deploying your dApp.
