# Authentication

Our authentication system is designed to be secure and flexible, offering several ways to log in to your account.

## Supported Authentication Methods

*   **Google**: Log in using your Google account for a fast and secure experience.
*   **Email and Password**: Create an account using your email address and a secure password.
*   **GitHub**: Log in using your GitHub account, which is a great option for developers.

## Security

All authentication methods are secured using **Firebase Authentication**, which provides a robust and reliable security framework.
